    • Input Format

    1. The input is expected to be well formed

      2. Braces covering the complete expression need not be there

    2. Brackets for negation operator are not required.

      3. Ex. ~p is valid

      Ex. ~(p^q) is valid

    The program may give a warning if a well bracketed input is not entered

    This is a segmentation fault handled by csignal (signal handling)

    Ubuntu version 16.04.2
